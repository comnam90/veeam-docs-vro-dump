# Configuring Plan Steps


Plan steps are the sequence of actions taken by the Orchestrator server during plan execution. For each machine in an inventory group included in a recovery plan, 2 types of steps can be performed:

* [Default steps provided by Veeam](steps_available.md)
* [Custom steps added by end user](adding_custom_scripts.md)

For detailed description of all available recovery plan steps, see [Appendix A. Recovery Plan Steps](appendix_plan_steps.md).

|  |
| --- |
| Note |
| You cannot delete built-in steps provided by Veeam. This option is available for [custom steps](adding_custom_scripts.md) only. |
