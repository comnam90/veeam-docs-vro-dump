# Creating Restore Plans


To create a restore plan:

1. Navigate to Recovery Plans.
2. Click Add New Plan.
3. Complete the New Recovery Plan wizard:

1. [Choose a type of the plan](restore_plan_type.md).
2. [Choose the necessary backup type](restore_plan_backup_type.md).
3. [Choose a scope for the plan](restore_plan_scope.md).
4. [Specify a plan name and description](restore_plan_name.md).
5. [Specify the target RTO and RPO](restore_plan_rto_rpo.md).
6. [Select a recovery location for the plan](restore_plan_location.md).
7. [Select a template for plan reports](restore_plan_report_template.md).
8. [Finish working with the wizard](restore_plan_finish.md).
