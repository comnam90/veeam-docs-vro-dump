# Exceeding License Limit


In some situations, the number of actually managed machines may exceed the license limit. For example, this may happen when some machines are temporarily used for testing.

To deal with a situation where you need to manage more machines than covered by your license, Orchestrator provides mechanisms of Allowed Increase Limit and New VMs.
