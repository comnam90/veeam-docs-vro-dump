# Scanning CDP Replica Plans


You can start on-demand plan scanning for malware and configure scan scheduling for any CDP replica plan. There is almost no difference between the procedures performed for replica, CDP replica, restore and cloud plans. For more information, see [Scanning Recovery Plans](scanning_recovery_plans.md).
