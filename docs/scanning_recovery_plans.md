# Scanning Recovery Plans


You can check a recovery plan for possible malware before or when running a plan.

|  |
| --- |
| Important |
| * To scan CDP replica plans for malware, the Veeam Backup & Replication server that manages replication jobs must run version 13.0.1 or later. * Scanning storage plans for possible malware is not supported. |
