# Adding Storage Recovery Locations


To add a storage recovery location:

1. Switch to the Administration page.
2. Navigate to Recovery Locations.
3. Click Add.
4. Complete the New Recovery Location wizard:

1. [Specify a recovery location name and description](storage_location_name.md).
2. [Choose a recovery location type](storage_location_type.md).
3. [Choose a storage vendor](storage_location_vendor.md).
4. [Specify storage and compute resources](storage_location_compute_resources.md).
5. [Configure network mapping and re-IP rules](storage_location_network_mapping.md).
6. [Finish working with the wizard](storage_location_finish.md).
