# Updating License


When your Orchestrator license expires, you must update it. To update the license, you can use either of the following methods:

* [Update the license manually](updating_license_manually.md)
* [Update the license automatically](updating_license_automatically.md)

|  |
| --- |
| Note |
| You can update the license automatically only if you have a paid license (Rental, Perpetual or Subscription) installed. |
