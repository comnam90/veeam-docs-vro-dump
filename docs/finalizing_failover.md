# Finalizing Failover


Orchestrator provides you with a number of options to finalize failover to VM replicas:

* [Perform permanent failover](running_permanent_failover.md)
* [Perform failback](running_failback.md)
* [Commit failback](committing_failback.md)

For more information on the available options, see [Failover and Failback](failover_failback_replica.md).
