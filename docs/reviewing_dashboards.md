# Reviewing Dashboards


Orchestrator comes with 2 dashboards that allow you to:

* Track the health state of the connected infrastructure. For more information, see [Administration Dashboard](administration_dashboard.md).
* Analyze the readiness for disaster recovery operations across different scopes. For more information, see [Home Page Dashboard](home_dashboard.md).
