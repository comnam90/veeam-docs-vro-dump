# Adding VMware vSphere Recovery Locations


To add a VMware vSphere recovery location:

1. Switch to the Administration page.
2. Navigate to Recovery Locations.
3. Click Add.
4. Complete the New Recovery Location wizard:

1. [Specify a recovery location name and description](restore_location_name.md).
2. [Choose a recovery location type](restore_location_type.md).
3. [Choose recovery options](restore_location_recovery_options.md).
4. [Specify compute resources](restore_location_compute_resources.md).
5. [Specify storage resources](restore_location_storage_resources.md).
6. [Configure network mapping and re-IP rules](restore_location_network_mapping.md).
7. [Finish working with the wizard](restore_location_finish.md).
