# Running Storage Failover


The Run action causes VMs in a plan to fail over to destination (NetApp) or secondary (HPE) storage volumes. For more information on the data recovery process, see the [NetApp ONTAP Documentation Center](https://docs.netapp.com/ontap-9/index.jsp?topic=%2Fcom.netapp.doc.pow-dap%2FGUID-EE407A7C-DF1B-47B3-9D9B-ACF780F846D2.html) and [Hewlett Packard Enterprise Support Center](https://support.hpe.com/hpesc/public/home).
