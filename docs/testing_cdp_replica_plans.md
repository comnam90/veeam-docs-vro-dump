# Testing CDP Replica Plans


You can start on-demand plan testing and configure test scheduling for any replica plan. There is almost no difference between the procedures performed for replica, CDP replica, restore and storage plans. For more information, see [Testing Recovery Plans](testing_recovery_plans.md).
