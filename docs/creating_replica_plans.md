# Creating Replica Plans


To create a replica plan:

1. Navigate to Recovery Plans.
2. Click Add New Plan.
3. Complete the New Recovery Plan wizard:

1. [Choose a type of the plan](replica_plan_type.md).
2. [Choose the necessary replica type](replica_plan_replica_type.md).
3. [Choose a scope for the plan](replica_plan_scope.md).
4. [Specify a plan name and description](replica_plan_name.md).
5. [Specify the target RTO and RPO](replica_plan_rto_rpo.md).
6. [Select a template for plan reports](replica_plan_report_template.md).
7. [Finish working with the wizard](replica_plan_finish.md).
