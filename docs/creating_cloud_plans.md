# Creating Cloud Plans


To create a cloud plan:

1. Navigate to Recovery Plans.
2. Click Add New Plan.
3. Complete the New Recovery Plan wizard:

1. [Choose a type of the plan](cloud_plan_type.md).
2. [Choose a type of backup files.](cloud_plan_backup_type.md)
3. [Choose a scope for the plan](cloud_plan_scope.md).
4. [Specify a plan name and description](cloud_plan_name.md).
5. [Specify the target RTO and RPO](cloud_plan_rto_rpo.md).
6. [Choose a recovery location](cloud_recovery_location.md).
7. [Select a template for plan reports](cloud_plan_report_template.md).
8. [Finish working with the wizard](cloud_plan_finish.md).
