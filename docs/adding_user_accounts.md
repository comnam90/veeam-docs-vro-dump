# Adding User Accounts


To add a user or a user group:

1. [Launch the Add New Account wizard](adding_users_launch.md).
2. [Specify an account to which you want to assign a role](adding_users_account.md).
3. [Assign roles and scopes to your account](adding_users_scopes.md).
4. [Finish working with the wizard](adding_users_finish.md).
