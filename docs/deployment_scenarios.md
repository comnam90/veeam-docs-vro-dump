# Deployment Scenarios


Orchestrator supports a number of deployment scenarios. This document describes 6 common scenarios.

* [Orchestrating restore to Microsoft Azure from vSphere VM and Veeam agent backups](infrastructure_restore_azure.md)
* [Orchestrating restore to Microsoft Hyper-V from vSphere and Hyper-V VM backups](infrastructure_hyperv.md)
* [Orchestrating restore to VMware vSphere from vSphere VM and Veeam agent backups](infrastructure_restore.md)
* [Orchestrating failover based on storage replication and vSphere VMs](infrastructure_storage_failover.md)
* [Orchestrating failover based on Veeam vSphere replication jobs](infrastructure_failover.md)
* [Orchestrating restore and VM replica failover in clean room](infrastructure_clean_room.md)

You can also combine any of the described scenarios to support your own data protection strategy.
