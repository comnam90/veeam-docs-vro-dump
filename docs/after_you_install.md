# After You Install


To start working with the Orchestrator UI, you must perform the initial configuration of the Orchestrator server.

1. Log in to the Orchestrator UI as the local Administrator who performed Orchestrator installation. For more information, see [Accessing Orchestrator UI](accessing_vro_ui.md).
2. [Launch the Initial Configuration Wizard](initial_config_dr_wizard.md).
3. [Configure server settings and deploy Orchestrator agents](initial_config_dr_details.md).
4. [Review configuration summary](initial_config_dr_summary.md).
