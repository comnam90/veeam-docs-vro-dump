# Configuring Default Parameter Settings


To modify default parameter settings for a plan step:

1. Switch to the Administration page.
2. Navigate to Recovery Steps.
3. In the Step column, select the step and click Edit.
4. In the Parameters section of the Step Editor window, set the desired parameter values and click Save.

|  |
| --- |
| Note |
| The parameter settings will be changed only for plans that are NOT in the IN-USE mode. For the list of modes that a recovery plan can acquire, see [Replica Plans](states_modes_replica.md), [CDP Replica Plans](states_modes_cdp.md), [Restore Plans](states_modes_restore.md), [Storage Plans](states_modes_storage.md) and [Cloud Plans](states_modes_cloud.md). |
